# 🏠 GK Landlords Realty — Official Website

A professional real estate website built for **GK Landlords Realty**, a property business based in Solapur, Maharashtra. Built as a real client project for a family business.

## ✨ Features
- Hero section with property stats & CTA
- Flagship projects showcase (Balaji Regency, Padmavati Regency, Samarth Villa, Kulaswamini Vihar)
- Filterable property listings (Sale / Rent / Plots)
- Contact form integrated with **Formspree**
- Smooth scroll animations with Intersection Observer
- Fully mobile-responsive design

---

## 🛠️ Tech Stack
- **HTML5** — Structure & layout
- **CSS3** — Custom properties, animations, responsive grid
- **Vanilla JavaScript** — Filter logic, scroll reveal
- **Formspree** — Contact form backend
- **Google Fonts** — Playfair Display + Jost
