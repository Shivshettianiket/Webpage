# first webpage
GK Landlords Realty Website

Project Description

This project is a real estate business website developed for GK Landlords Realty.
The website provides information about properties, services, and contact details for clients who are looking to buy, sell, or rent properties.

Features

- Responsive real estate website design
- Property information section
- Business introduction page
- Contact form for customer enquiries
- Mobile-friendly layout

Technologies Used

- HTML
- CSS
- JavaScript

Installation / How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open the "index.html" file in your browser.

Purpose of the Project

This website was created to help GK Landlords Realty promote their real estate business online and make it easier for customers to contact them.
